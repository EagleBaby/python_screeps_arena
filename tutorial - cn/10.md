在本节教程中，您需要创建Creep抵御敌人的攻击。游戏中提供了一个友方的`Spawn`和距离2的一个`Source`，以及3个敌方`Creep`，其大约会在360tick左右抵达你的Spawn，配方分别为`MA` `MH` `MR`。

### 获取对象

首先获取所需对象：

```python
from builtin import *

SPAWN = get.spawn()
SOURCE = get.source()
ENEMIES = get.creeps(st.enemy)
WALL = get.wall(6, 6)

WORRIER = None
```

### 速攻策略

速攻策略是最简单直接的方案，无需开局生产，直接创建攻击单位，先破墙，再消灭敌人。

### 最终代码

以下是速攻策略的完整代码：

```python
from builtin import *

SPAWN = get.spawn()
SOURCE = get.source()
ENEMIES = get.creeps(st.enemy)
WALL = get.wall(6, 6)

WORRIER = None


def init(k: GlobalKnowledge):
    pass


def step(k: GlobalKnowledge):
    global WORRIER
    if not st.creep(WORRIER):
        WORRIER = SPAWN.create("M4A4, "Worrier1", True, LEFT)
    # Worrier
    if not st.creep(WORRIER): return
    if WALL.exists:
        WORRIER.attack(WALL)
    else:
        enemy = WORRIER.closest(ENEMIES)
        if enemy: WORRIER.attack(enemy)
```
