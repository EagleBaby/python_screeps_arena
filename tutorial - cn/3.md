
## 3. 攻击敌人
  - [获取对象](#获取对象)
  - [攻击指令](#攻击指令)
  - [最终代码](#最终代码)

**目标**：控制友方 Creep 攻击并消灭敌方 Creep。

### 获取对象
使用 `get` 获取游戏对象，结合 `st` 模块的判断函数区分敌我：

- `get.游戏对象名(可选筛选函数)` - 获取一个特定游戏对象
- `st.my(obj)`/`st.friend(obj)` - 判断是否为己方单位
- `st.enemy(obj)` - 判断是否为敌方单位

**示例**：
```python
from builtin import *

# 获取己方 Creep 和敌方 Creep
CREEP = get.creep(st.friend)
ENEMY = get.creep(st.enemy)
```

### 攻击指令

`creep.attack(target: st.hitable, options: MotionOptions | bool = True) -> int`

| 参数名 | 类型 | 可选 | 默认值 | 描述 |
|-------|------|------|--------|------|
| `target` | `st.hitable` | 否 | - | 目标对象（具有 hits 和 hitsMax 属性的对象） |
| `options` | `MotionOptions` \| `bool` | 是 | `True` | 移动控制参数 |

**返回值**：错误码
- `OK` (0): 操作成功
- `ERR_NOT_OWNER` (-1): 非 Creep 拥有者
- `ERR_INVALID_ARGS` (-10): 参数不正确
- `ERR_NO_BODYPART` (-12): 无 ATTACK 或 RANGED_ATTACK 部件

**移动控制**：
- `True` 或 `MotionOptions` 对象：自动靠近/远离目标（远程单位会保持距离）
- `False`：禁用自动移动，需要手动控制

### 最终代码
```python
from builtin import *

CREEP = get.creep(st.friend)
ENEMY = get.creep(st.enemy)

def init(k: GlobalKnowledge):
    pass

def step(k: GlobalKnowledge):
    CREEP.attack(ENEMY)
```
