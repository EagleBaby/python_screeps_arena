

## 2. 对象获取与移动
  - [获取对象](#获取对象)
  - [移动指令](#移动指令)
  - [最终代码](#最终代码)

**目标**：获取Creep和Flag对象，控制Creep移动到Flag处。

### 获取对象
使用`get`获取游戏对象：

- `get.creep()` - 获取一个Creep
- `get.flag()` - 获取一个Flag
...
- `get.游戏对象名()` - 相似的，获取其他游戏对象
...
- `get.游戏对象名s()` - 获取多个对象，返回值是list

**示例**：
```python
from builtin import *

# 全局变量存储对象引用
CREEP = get.creep()
FLAG = get.flag()
```

### 移动指令

`creep.move(to: Point | int, options: MotionOptions = None) -> int`

| 参数名 | 类型 | 可选 | 默认值 | 描述 |
|-------|------|------|--------|------|
| `to` | `Point`/`int` | 否 | - | 目标位置或方向常量 |
| `options` | `MotionOptions` | 是 | `None` | 寻路选项 |

**返回值**：错误码
- `OK` (0): 操作成功
- `ERR_NOT_OWNER` (-1): 非Creep拥有者
- `ERR_INVALID_ARGS` (-10): 方向不正确
- `ERR_TIRED` (-11): 疲劳值不为零
- `ERR_NO_BODYPART` (-12): 无MOVE部件

**预设选项**：
- `DEFAULT_MOTION` - 默认选项
- `SWAMP_MOTION` - 忽略沼泽地形

### 最终代码
```python
from builtin import *

CREEP = get.creep()
FLAG = get.flag()

def init(k: GlobalKnowledge):
    pass

def step(k: GlobalKnowledge):
    CREEP.move(FLAG)
```
