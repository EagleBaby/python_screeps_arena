This chapter introduces three output methods in Screeps Arena and the code deployment process.

### Print
**Basic output, only effective when tick >= 1.**
```python
from builtin import *


# ❌ Invalid in global area, won't be displayed
print("Hello, at global.")   # tick == 0


def init(k: GlobalKnowledge):
    print("Hello, at init.")   # tick == 1


def step(k: GlobalKnowledge):
    print("Hello, at step.")   # tick >= 1
```

### Jprint
<details>
<summary>
For debugging, displays JavaScript native object structure.
</summary>

Comparison with `print`:

| Data Type | print | jprint |
|----------|-------|--------|
| Simple types (int/list) | `1 [1,2]` | `1 [1,2]` |
| Object | Formatted string | Complete property list |

**Use case**: Enable when you need to view all details of an object.

</details>

### Standard Output
<details>
<summary>
Graded logging, indicating the caller.
</summary>

**General parameter format**: `(caller: str, msg: str)`
- `caller`: Caller name, used to distinguish logs from different sources
- `msg`: Message content to output

```python
std.log("caller", "normal log")    # Gray
std.info("caller", "info message")   # Green
std.warn("caller", "warning message")   # Yellow
std.error("caller", "fatal error")  # Red and interrupt execution
```

</details>

### Compilation and Execution

**Three steps to complete deployment:**

1. **Configure `build.py`**
   - In-game → `External Editor` → Right-click `main.mjs` → Copy path
   - Paste into `build.py`'s `config.target` (uncomment)
   - Optional: Set `config.language='cn'` to enable partial Chinese localization

2. **Execute compilation**
   - Run `build.py` in PyCharm
   - Python code is automatically transpiled to JavaScript

3. **Verify results**
   - Click `Play Game`
   - Success → Pop up `Pass` prompt

**Common issues**:
- `[Compiler.compile Error]: Target export directory does not exist`
  - Solution: Ensure `config.target` path is correct and directory exists.
