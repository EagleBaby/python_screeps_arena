
## 3. Attacking Enemies

  - [Final Code](#final-code)

**Objective**: Control friendly Creep to attack and eliminate enemy Creep.

### Acquiring Objects
Use `get` to acquire game objects, combined with `st` module's judgment functions to distinguish friend from foe:

- `get.game_object_name(optional filter function)` - Get a specific game object
- `st.my(obj)` / `st.friend(obj)` - Determine if it's a friendly unit
- `st.enemy(obj)` - Determine if it's an enemy unit

**Example**:
```python
from builtin import *

# Get friendly Creep and enemy Creep
CREEP = get.creep(st.friend)
ENEMY = get.creep(st.enemy)
```

### Attack Commands

`creep.attack(target: st.hitable, options: MotionOptions | bool = True) -> int`

| Parameter Name | Type | Optional | Default Value | Description |
|---------------|------|----------|---------------|-------------|
| `target` | `st.hitable` | No | - | Target object (object with hits and hitsMax properties) |
| `options` | `MotionOptions` \| `bool` | Yes | `True` | Movement control parameter |

**Return Value**: Error code
- `OK` (0): Operation successful
- `ERR_NOT_OWNER` (-1): Not Creep owner
- `ERR_INVALID_ARGS` (-10): Incorrect parameters
- `ERR_NO_BODYPART` (-12): No ATTACK or RANGED_ATTACK part

**Movement Control**:
- `True` or `MotionOptions` object: Automatically approach/retreat from target (ranged units will maintain distance)
- `False`: Disable automatic movement, manual control required

### Final Code
```python
from builtin import *

CREEP = get.creep(st.friend)
ENEMY = get.creep(st.enemy)

def init(k: GlobalKnowledge):
    pass

def step(k: GlobalKnowledge):
    CREEP.attack(ENEMY)
```
