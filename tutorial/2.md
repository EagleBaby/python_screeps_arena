**Objective**: Acquire Creep and Flag objects, control Creep to move to Flag.

### Acquiring Objects
Use `get` to acquire game objects:

- `get.creep()` - Get a Creep
- `get.flag()` - Get a Flag
...
- `get.game_object_name()` - Similarly, get other game objects
...
- `get.game_object_names()` - Get multiple objects, return value is a list

**Example**:
```python
from builtin import *

# Global variables store object references
CREEP = get.creep()
FLAG = get.flag()
```

### Movement Commands

`creep.move(to: Point | int, options: MotionOptions = None) -> int`

| Parameter Name | Type | Optional | Default Value | Description |
|---------------|------|----------|---------------|-------------|
| `to` | `Point` / `int` | No | - | Target position or direction constant |
| `options` | `MotionOptions` | Yes | `None` | Pathfinding options |

**Return Value**: Error code
- `OK` (0): Operation successful
- `ERR_NOT_OWNER` (-1): Not the Creep owner
- `ERR_INVALID_ARGS` (-10): Incorrect direction
- `ERR_TIRED` (-11): Fatigue value not zero
- `ERR_NO_BODYPART` (-12): No MOVE part

**Preset options**:
- `DEFAULT_MOTION` - Default options
- `SWAMP_MOTION` - Ignore swamp terrain

### Final Code
```python
from builtin import *

CREEP = get.creep()
FLAG = get.flag()

def init(k: GlobalKnowledge):
    pass

def step(k: GlobalKnowledge):
    CREEP.move(FLAG)
```
