In this tutorial, you need to create Creeps to defend against enemy attacks. The game provides a friendly `Spawn`, a `Source` at distance 2, and 3 enemy `Creeps` that will reach your Spawn around tick 360, with recipes `MA` `MH` `MR`.

### Acquiring Objects

First acquire required objects:

```python
from builtin import *

SPAWN = get.spawn()
SOURCE = get.source()
ENEMIES = get.creeps(st.enemy)
WALL = get.wall(6, 6)

WORRIER = None
```

### Quick Attack Strategy

Quick attack strategy is the simplest and most direct approach, no initial production needed, directly create attack units, break walls first, then eliminate enemies.

### Final Code

Complete code for quick attack strategy:

```python
from builtin import *

SPAWN = get.spawn()
SOURCE = get.source()
ENEMIES = get.creeps(st.enemy)
WALL = get.wall(6, 6)

WORRIER = None


def init(k: GlobalKnowledge):
    pass


def step(k: GlobalKnowledge):
    global WORRIER
    if not st.creep(WORRIER):
        WORRIER = SPAWN.create("M4A4", "Worrier1", True, LEFT)
    # Worrier
    if not st.creep(WORRIER): return
    if WALL.exists:
        WORRIER.attack(WALL)
    else:
        enemy = WORRIER.closest(ENEMIES)
        if enemy: WORRIER.attack(enemy)
```
